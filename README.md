# Interview

Реализации функций можно найти в src/generators.py

## installation 
```
    $ git clone git@github.com:renshs/interview.git
    $ cd interview/
    $ python3 -m venv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
```

## running
```
    $ python3 main.py
```

## running tests
```
    $ pytest -v .
```
